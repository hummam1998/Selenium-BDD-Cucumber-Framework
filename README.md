# Selenium Cucumber BDD Framework

## Overview
This is a Selenium + Cucumber BDD automation testing framework designed for web application testing using Java and Maven.

## Tech Stack
- Java
- Selenium WebDriver
- Cucumber (BDD)
- Maven

## Prerequisites
- Java 8+
- Maven
- IDE (IntelliJ / Eclipse)

## Project Structure
- src/test/java – Step definitions & runners
- src/test/resources – Feature files
- pom.xml – Dependency management

## Run Tests
mvn clean test

## Reports
Cucumber reports are generated in the target directory.

## CI/CD
Framework is CI-ready (Jenkins / GitHub Actions).
